# code-con
> A Docker container image for VS Code and CDR code-server

## Motivation

|     App     |           Pros           |                Cons                |
|:-----------:|:------------------------:|:----------------------------------:|
|     Code    | Full IDE with extensions |             Desktop app            |
| code-server | Web app                  | Cannot install MS extensions (TOS) |
